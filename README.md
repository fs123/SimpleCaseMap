# Simple example of a case map, implemented with react and redux

## Try it 
Preconditions: NodeJS 6.x

```sh
$ git clone https://github.com/fs123/SimpleCaseMap
$ cd SimpleCaseMap
$ npm install
$ npm run dev
```
